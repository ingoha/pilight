.. rubric:: Supported Brands

Shelly

.. rubric:: Sender Arguments

.. code-block:: console
   :linenos:

   -i --id=id             control a device with this id
   -t --on                send an on signal
   -f --off               send an off signal

.. rubric:: Config

.. code-block:: json
   :linenos:

   {
     "devices": {
       "switch": {
         "protocol": [ "shelly1" ],
         "id": [{
           "id": "A01BC2",
         }],
         "state": "off"
       }
     },
     "gui": {
       "switch": {
         "name": "Switch",
         "group": [ "Living" ],
         "media": [ "all" ]
       }
     }
   }

+------------------+-----------------+
| **Option**       | **Value**       |
+------------------+-----------------+
| id               | as hardcoded    |
+------------------+-----------------+
| state            | on / off        |
+------------------+-----------------+
